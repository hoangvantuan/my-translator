Behavior

  • Font size: A−/A+ thay đổi realtime
  • Color: click dot → apply ngay, active dot có border
  • Opacity: drag slider → preview realtime, debounce save

4. Visibility & Position

Position

  • Bottom-right của overlay window
  • Giữ vị trí bottom: 24px; right: 16px
  • Popover mở phía trên bar

Hover behavior

  • Ẩn mặc định (opacity: 0)
  • Show khi hover overlay (giữ delay 1.5s)
  • Popover đang mở → bar không ẩn

Xoá bỏ

  • Xoá hoàn toàn #floating-toolbar (HTML + CSS + JS)
  • Xoá #btn-view-mode duplicate
  • Xoá CSS .floating-toolbar

Giữ lại

  • Keyboard shortcuts (⌘1/2/3, ⌘T, Space)
  • Button IDs cho app.js handlers
  • Settings sync, debounce logic
